# GIT COMMIT TYPE LIST 
# Feat : 새로운 기능 추가
# Fix : 버그 및 오류 수정
# Docs : 문서 수정
# Refact : 코드 리팩토링
# Style : 코드 의미에 영향을 주지 않는 변경사항
# Test : 테스트 코드 추가
# Chore : 빌드 부분 혹은 패키지 매니저 수정사항
